Glaxy_ShinX取扱説明書

1.概要

この度は、galaxy_ShinXをダウンロードしていただきありがとうございます。
このアプリは、球をよけるゲームとなっています。尚、本バージョン'ver.0.1α'はα版となっておりますので、バグ・改善点等たくさんあると思いますのでその節はぜひご意見等お願い致します。プレイされた後はぜひ下記リンクにてアンケートにご協力お願い致します。 アンケートページです

2.操作方法

本ゲームは誰でも操作が出来るように、簡単な操作方法となっております。操作方法は以下の通りです。
←左へ移動:→右へ移動
となっております

3.動作環境

本ゲームの動作環境は以下の通りです。※著者が試した環境です。ほかの環境でも動く可能性があります。
OS:windows10
CPU:各OSが動くもの
RAM:512MB以上
アーキテクチャ:32bit/64bit
GPU:direct9以上

4.ルール

上から降ってくる球を、矢印キーを使いよけます。そして、最高スコアを目指していきます。頑張ってください!!
